Welcome to conso

Two quick permissions and you're ready. conso checks these live every launch — it never assumes a grant is still valid.

Full Disk Access Not granted Required to scan, analyze and clean protected folders. We verify it by actually reading a protected path — not by trusting the toggle.
Privileged Helper Not installed Lets maintenance & deep-clean run with admin rights. Installed once via SMAppService — one password, then never again.
Notifications Optional A heads-up when a cleanup or update finishes.
conso re-checks on every launch and points you to the exact fix if anything's missing.
If macOS blocks the first launch ("unidentified developer"), open System Settings → Privacy & Security → Open Anyway. The old right-click → Open shortcut was removed in macOS 15.
conso processes everything locally. Nothing leaves your Mac.
